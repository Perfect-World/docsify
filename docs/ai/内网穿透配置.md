## 服务端

### 进入跟目录
cd /mydata/frp

### 下载服务端
https://github.com/fatedier/frp/releases
wget https://github.com/fatedier/frp/releases/download/...

### 解压文件夹 切至该目录
tar -zxvf frp_ & cd frp_

### 编辑frps.toml配置管理端(服务管理端)
vi frps.toml

```sh
# bindPort就是客户端待会要链接的端口
bindPort = 50004
# frp 服务端监听的 HTTP 虚拟主机端口，让 frp 服务器在 50006 端口接收 HTTP 域名请求
vhostHTTPPort = 50006
# 默认为 127.0.0.1，如果需要公网访问，需要修改为 0.0.0.0
webServer.addr = "0.0.0.0"
webServer.port = 50005
# dashboard 用户名密码，可选，默认为空
webServer.user = "admin"
webServer.password = "admin"
```

### systemd启动服务端（官网推荐）

#### 安装 systemd

#### 使用 yum 安装 systemd（CentOS/RHEL）
yum install systemd

#### 使用 apt 安装 systemd（Debian/Ubuntu）我使用的这个命令，根据自己系统来选择
apt install systemd

### 创建 frps.service 文件
使用文本编辑器 (如 vim) 在 /etc/systemd/system 目录下创建一个 frps.service 文件，
用于配置 frps 服务。
sudo vim /etc/systemd/system/frps.service
```sh
[Unit]
# 服务名称，可自定义
Description = frp server
After = network.target syslog.target
Wants = network.target

[Service]
Type = simple
# 启动frps的命令，需修改为您的frps的安装路径  注意这里要修改/path/to为你的文件目录！！！
ExecStart = /path/to/frps -c /path/to/frps.toml

[Install]
WantedBy = multi-user.target
```
### 启动frp
sudo systemctl start frps
### 停止frp
sudo systemctl stop frps
### 重启frp
sudo systemctl restart frps
### 查看frp状态
sudo systemctl status frps
### 设置 frps 开机自启动
sudo systemctl enable frps
### 看一下服务端启动的效果 如果访问不通的话，可以先服务器上 telnet localhost 7500
访问地址：http://121.237.178.252:50005/static/#/
### 查看所有 frps 相关端口
netstat -tulpn | grep frps


## 客户端

### mac

#### 下载客户端

wget https://github.com/fatedier/frp/releases/download/...

#### 解压文件夹 切至该目录

tar -zxvf frp_ & cd frp_

#### 编辑frps.toml配置客户端(本地电脑、内网服务器)

vi frpc.toml

```sh
serverAddr = "121.237.178.252"
serverPort = 50004

[[proxies]]
name = "mac_connect"
type = "tcp"
localIP = "127.0.0.1"
localPort = 5900
# 外网访问你内网 VNC 的端口（别人连你用）
remotePort = 5900
```

#### 启动客户端
sudo ./frpc -c ./frpc.toml
